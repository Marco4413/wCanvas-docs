    Rectangles' config

    Properties:
    Properties
    Name Type Attributes Description
    noStroke Boolean <optional>

    Whether or not stroke should be applied
    noFill Boolean <optional>

    Whether or not the shape should be filled
    alignment Alignment <optional>

    Specifies the position of the origin of the rectangle
    rounded Object <optional>

    If not undefined the rectangle will be drawn as if it has round corners and its x and y will be the center of it
    Name Type Attributes Description
    corners Array.<Boolean> <optional>

    Corners that should be drawn (clockwise starting from the top-left one), if undefined all corners will be drawn
    radiusMode "percentage" | "pixels" <optional>

    How RectConfig#rounded.radius should be handled, if undefined it defaults to "percentage".
    "percentage": Takes the smallest between the width and the height and uses it to calculate the radius of the corner in pixels;
    "pixels": It uses the specified number as the radius.
    radius Number <optional>

    Its behaviour depends on RectConfig#rounded.radiusMode
